# 📝 Notes Management API

A REST API built with FastAPI for secure note management.

## Features
- User Signup & Login
- JWT Authentication
- Password Hashing
- CRUD Operations
- Search Notes
- Pagination
- Filtering
- Role-Based Access Control

## Technologies
- Python
- FastAPI
- SQLAlchemy
- Pydantic
- JWT
- SQLite

## Run the Project

```bash
pip install -r requirements.txt
uvicorn main:app --reload
